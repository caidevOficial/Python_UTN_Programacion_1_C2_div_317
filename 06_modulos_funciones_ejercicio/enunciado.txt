Crear una aplicacion "CalcUTN" la cual tenga un menú que sirva para acceder a cualquier opción que elija el usuario.

Las opciones del menú deben ser:

01 - Sumar dos números
02 - Restar dos números
03 - Multiplicar dos números
04 - Dividir dos números
05 - Potencia de un número
06 - Saber número primo
07 - Números primos hasta
08 - El área de un circulo
09 - El área de un rectángulo
10 - El área de un cuadrado
11 - Salir

Al terminar la ejecución de alguna acción, la aplicación debe volver al 
menú principal para darle la oportunidad al usuario 
de elegir otra opción o salir.

IMPORTANTE: Usa módulos y funciones para resolverlo.
